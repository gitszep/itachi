Połącz się ze zdalnym repozytorium za pomocą polecenia:

git remote add origin url

Utwórz nową gałąź roboczą i przełącz się na nią:

git checkout -b galaz

Wprowadź zmiany w projekcie.

Dodaj zmienione pliki do indeksu:

git add .

Zatwierdź zmiany z odpowiednim opisem:

git commit -m "opis zmian"

Przełącz się na gałąź główną (main):

git checkout main

Scal swoją gałąź roboczą z gałęzią główną:

git merge galaz

Dodaj i zatwierdź ewentualne dodatkowe zmiany:

git add .
git commit -m "opis zmian"

Pobierz najnowsze zmiany z repozytorium zdalnego:

git pull origin main

Wypchnij lokalne zmiany do repozytorium zdalnego:

git push origin main

Jeżeli nie możesz zmergować

Rozwiązanie siłowe (jeśli nie masz tych zmian w main, a chcesz je tam wrzucić):

# Wejdź na itachi
git checkout itachi

# Skopiuj plik
cp index.html /tmp/index.html

# Wejdź na main
git checkout main

# Nadpisz plik z backupu
cp /tmp/index.html index.html

# Zapisz zmianę
git add index.html
git commit -m "Wrzucenie kodu HTML z itachi na main"
